# Установка

```
npm i
```


# Использование

```
npm start
```

Проект доступен по `http://localhost:3000`


# Описание

## Структура папок

```
./dist                 — готовый шаблон
./dist/fonts           — шрифты
./dist/html            — шаблоны
./dist/images          — графика
./dist/scripts         — js
./dist/styles          — css

./src                  — исходники
./src/html             — шаблоны
./src/scripts          — js
./src/scripts/vendor   - всё, что в этой папке копируются целиком в dist/...
./src/styles           — scss bootstrap4 и blocks для .styl

./tasks                – задачи для gulp
```
